Running program in VSCODE instructions:

1. Terminal > New Terminal
2. Terminal > Split Terminal
3. In each terminal: cd "Project Code"
4. Left Terminal: javac Client.java (Creates Client.class)
5. Right Terminal: javac Server.java (Creates Server.class and Server$Handler.class)
6. Right Terminal: java Client
7. Left Terminal: java Server
8. Control + C in each terminal to end program
